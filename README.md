## Bellabeat Marketing Analysis Case Study

## Business Objective
Analyze smart device usage data to identify trends and provide data-driven marketing recommendations for Bellabeat.

## Data Overview
- Multiple Fitbit datasets provided
- Selected relevant daily activity and sleep datasets
- Excluded incomplete and low-usage user records

## Data Cleaning & Preparation
- Removed duplicates
- Standardized date formats
- Filtered users with fewer than 10 active days
- Joined activity and sleep datasets

## Key Insights
- Users with lower activity levels tend to have longer sleep duration
- Highly active users sleep fewer minutes on average
- Activity patterns vary significantly by day of week

## Tools Used
- SQL (Google BigQuery)
- Tableau (data visualization)
- GitHub (version control)

## Marketing Recommendations
- Emphasize sleep quality and recovery for active users
- Schedule engagement campaigns around peak activity days
- Use personalized wellness messaging
